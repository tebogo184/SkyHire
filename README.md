# SkyHire

SkyHire is a full-stack **e-commerce platform** built with a **Spring Boot backend** and a **web frontend** integrated into a single repository.  
This repository serves as the **core backend system**, responsible for business logic, data management, and API communication with the frontend.

The project was developed to demonstrate practical software engineering skills, including backend development, REST API design, and full-stack integration.

---

##  Project Overview

SkyHire provides the backend foundation for an online shopping platform, handling:

- Product management
- User interactions
- Order processing
- Communication with a web frontend

The frontend is included inside this repository under the `/web` directory.

---

##  Technology Stack

### Backend

### Frontend
